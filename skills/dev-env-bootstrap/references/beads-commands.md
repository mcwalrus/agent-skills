# Beads / Dolt Commands

## Allow (when beads/ or .beads/ detected)
- `Bash(bd *)`
- `Bash(dolt status)`
- `Bash(dolt log *)`
- `Bash(dolt diff *)`
- `Bash(dolt add *)`
- `Bash(dolt commit *)`
- `Bash(dolt push *)`
- `Bash(dolt pull *)`
- `Bash(dolt branch *)`
- `Bash(dolt merge *)`
- `Bash(dolt checkout *)`
- `Bash(dolt sql *)`
- `Bash(dolt table *)`
- `Bash(dolt schema *)`
- `Bash(dolt backup *)`
- `Bash(dolt fetch *)`
- `Bash(dolt tag *)`
- `Bash(dolt remote *)`
- `Bash(dolt version)`

## Ask
- `Bash(dolt reset --hard *)`
- `Bash(dolt clean -f *)`

## Deny
- `Bash(dolt push --force *)`
