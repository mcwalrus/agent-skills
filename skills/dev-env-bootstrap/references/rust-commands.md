# Rust Commands

## Allow
- `Bash(cargo test *)`
- `Bash(cargo nextest *)`
- `Bash(cargo clippy *)`
- `Bash(cargo check *)`
- `Bash(cargo fmt *)`
- `Bash(cargo doc *)`
- `Bash(cargo build *)`
- `Bash(cargo audit *)`
- `Bash(cargo tree *)`
- `Bash(cargo bench *)`
- `Bash(rustfmt *)`

## Ask
- `Bash(cargo add *)`
- `Bash(cargo remove *)`
- `Bash(cargo update *)`
- `Bash(cargo install *)`
- `Bash(cargo publish *)`
- `Bash(rustup update *)`

## Deny
- `Bash(cargo clean --force *)`
