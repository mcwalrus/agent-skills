# Kubernetes Commands

## Allow
- `Bash(kubectl get *)`
- `Bash(kubectl describe *)`
- `Bash(kubectl logs *)`
- `Bash(kubectl top *)`
- `Bash(kubectl explain *)`
- `Bash(kubectl config view)`
- `Bash(kubectl config current-context)`
- `Bash(helm lint *)`
- `Bash(helm template *)`
- `Bash(helm search *)`
- `Bash(kubescape scan *)`

## Ask
- `Bash(kubectl apply *)`
- `Bash(kubectl create *)`
- `Bash(kubectl set *)`
- `Bash(kubectl rollout *)`
- `Bash(helm install *)`
- `Bash(helm upgrade *)`

## Deny
- `Bash(kubectl delete *)`
- `Bash(helm uninstall *)`
- `Bash(kubectl drain *)`
- `Bash(kubectl cordon *)`
