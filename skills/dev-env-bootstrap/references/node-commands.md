# Node.js Commands

## Allow (safe, read-only or local)
- `Bash(npm test *)`
- `Bash(npm run build *)`
- `Bash(npm run lint *)`
- `Bash(npm run format *)`
- `Bash(npm run dev *)`
- `Bash(npx *)` (for one-off binaries)
- `Bash(pnpm test *)`
- `Bash(pnpm run *)`
- `Bash(yarn test *)`
- `Bash(yarn build *)`
- `Bash(biome check *)`
- `Bash(biome format *)`
- `Bash(biome lint *)`
- `Bash(eslint *)`
- `Bash(prettier --check *)`
- `Bash(vitest *)`
- `Bash(jest *)`

## Ask (modifies shared state)
- `Bash(npm install *)`
- `Bash(npm publish *)`
- `Bash(npm update *)`
- `Bash(pnpm install *)`
- `Bash(pnpm update *)`
- `Bash(yarn install *)`
- `Bash(yarn upgrade *)`

## Deny (destructive)
- `Bash(npm unpublish *)`
- `Bash(npm run clean:force *)` (if it rm -rf's)
