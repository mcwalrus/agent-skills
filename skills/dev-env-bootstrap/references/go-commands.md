# Go Commands

## Allow
- `Bash(go test *)`
- `Bash(go test ./...)`
- `Bash(go build *)`
- `Bash(go vet *)`
- `Bash(go fmt *)`
- `Bash(go mod tidy)`
- `Bash(go mod verify)`
- `Bash(go run *)`
- `Bash(golangci-lint run *)`
- `Bash(gosec *)`
- `Bash(govulncheck *)`

## Ask
- `Bash(go get *)`
- `Bash(go mod download)`
- `Bash(go install *)`
- `Bash(go env *)`

## Deny
- `Bash(go clean -cache -testcache -modcache)` (can break builds)
