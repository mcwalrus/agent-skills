# Shell / General Commands

## Allow (universal safe commands)
- `Bash(ls *)`
- `Bash(git worktree list)`
- `Bash(cat *)`
- `Bash(grep *)`
- `Bash(find *)`
- `Bash(wc *)`
- `Bash(head *)`
- `Bash(tail *)`
- `Bash(awk *)`
- `Bash(sed *)`
- `Bash(command -v *)`
- `Bash(* --version)`
- `Bash(* --help)`
- `Bash(just test *)`
- `Bash(just lint *)`
- `Bash(just fmt *)`
- `Bash(just validate *)`
- `Bash(just ci *)`
- `Bash(just check *)`
- `Bash(just build *)`
- `Bash(just pre-commit *)`
- `Bash(make test *)`
- `Bash(make lint *)`
- `Bash(make format *)`
- `Bash(make check *)`
- `Bash(shellcheck *)`
- `Bash(bash -n *)`

## Ask
- `Bash(just deploy *)`
- `Bash(just release *)`
- `Bash(just sync *)`
- `Bash(make install *)`
- `Bash(mv *)`
- `Bash(cp *)`
- `Bash(rm *)`
- `Bash(mkdir *)`
- `Bash(git worktree add *)`

## Deny
- `Bash(git worktree remove *)`
- `Bash(rm -rf /*)`
- `Bash(rm -rf ~*)`
- `Bash(rm -rf /)`
- `Bash(sudo *)`
- `Bash(curl * | bash *)`
- `Bash(curl * | sh *)`
- `Bash(wget * | bash *)`
- `Bash(wget * | sh *)`
